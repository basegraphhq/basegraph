# Basegraph

Basegraph is a monorepo for tools that build property graphs from source code and help teams plan implementation work. Use this repository as a starting point—detailed setup and usage guides live within each project.

## Projects

- `codegraph/` – Go extraction engine and graph tooling.
  - Documentation: `codegraph/Readme.md`
- `relay/` – Planning agent service that produces implementation-ready specs.
  - Documentation: see the guides in `relay/docs/`
- `dashboard/` – Next.js dashboard for exploring graph data and running workflows.
  - Documentation: reference the materials in `dashboard/docs/`

## Additional Resources

- `docs/onboarding.md` – Onboarding notes for the monorepo
- `execution_trace.md` – Historical experiments and architectural context
